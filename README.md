# GUI Magazyn
prosta aplikacja pozwalająca na zarządzanie magazynem stworzona na potrzeby przedmiotu GUI na PJWSTK.


Aplikacja działa na kolekcjach i zapisuje stan do pliku.
(Nie ma możliwości odczytania stanu magazynu po wylaczeniu programu)


W przyszłości zostanie dodane połaczenie z baza danych bądź odczyt stanu z pliku.
